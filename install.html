Installation¶

To install all the necessary dependencies and install AVRS-web as a package, at the command line:

python setup.py install

However, if you want to install the necessary packages without install AVRS-web, simple use pip install:

pip install -r requirements.txt

Then you are good to go. To run the app, please check out the Cookbook.
